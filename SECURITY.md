# Security Policy

## Supported Versions

Only the latest version of this portfolio repository is actively supported and maintained. 

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover any security vulnerability in this project, please report it immediately by opening a private report or contacting the repository owner via their GitHub profile contact details at [Himanshukumar1306](https://github.com/Himanshukumar1306).

Please do not open a public GitHub issue for security issues. Once the report is received:

1. We will acknowledge receipt of the report within 48 hours.
2. We will investigate the issue and, if verified, prepare a fix.
3. A fix will be committed and deployed directly to the default branch.
